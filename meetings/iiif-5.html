IIIF 5: February 19-21 2014, Copenhagen

Dates and Location

The fifth meeting of the IIIF Consortium is to be held at the Royal Library in Copenhagen, February 19-21 2014.

Address

Royal Library of Denmark
Soren Kierkegaards Plads 1
Copenhagen
DENMARK

Agenda

  • Wednesday Morning
    • Demos of latest work from participants
    • Review IIIF website
  • Wednesday Afternoon
    • Survey Client Software
    • Tools and Utilities
    • Image API Breakout: Minor Updates
  • Thursday Morning
    • Image Server Implementation Status
    • Metadata API Breakout: Minor updates
  • Thursday Afternoon
    • Metadata API: New Features
    • IIIF Community Expansion
    • IIIF Governance & Futures
  • Friday Morning
    • Future Plans: Discovery, Search, Annotation Lifecycle, non-image content
  • Friday Afternoon
    • Breakout groups as needed
    • Wrap up
